# Solvera Energy CRM

A comprehensive customer relationship management system built for solar energy companies.

## Features

- **Customer Management** - Track customers, projects, and installations
- **Photo Documentation** - Site surveys, installations, and inspections
- **Document Management** - Store and organize project documents
- **Scheduling System** - Manage appointments and work tickets
- **Commission Tracking** - Automated sales commission calculations
- **Role-Based Access** - Admin, Sales Manager, Sales Rep, and Field Tech roles
- **Activity Logging** - Track all project activities
- **Project Chat** - Team communication with @mentions

## Technology Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Icons**: Lucide React
- **Deployment**: Netlify

- Supabase account

- Netlify account (for hosting)

## Deployment

1. Extract package
2. Install dependencies: `npm install`
3. Configure environment variables
4. Push to GitHub
5. Deploy to Netlify
6. Create admin user

Full instructions in `DEPLOYMENT_CHECKLIST.md`
